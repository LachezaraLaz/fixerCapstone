(async, inner) handleSignIn() → {Promise.<void>}

Description:

  • Handles the sign-in process for a client user.

    This function validates the email and password fields, sends a sign-in request to the server, and handles the response by storing tokens and navigating to the main tabs screen upon successful sign-in.
